## Running

Clone the repository.
Change directory into the cloned project.
Run the following command.

```sh
$ docker-compose up
```

Browse here,
```
http://localhost/
```

To run PHP scripts,
```
docker exec -it php7-dockerized_php_1 sh
cd default
php index.php
```
